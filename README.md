## アプリ作成の目的
自分が意識したことは「_自走力_」です。
エンジニアには「分からないことを解決する能力」が必要です。
そのためには「言語化力、質問力、検索力」など必要な能力は多岐に渡ります。
それらの力を養うためには、「_実務を意識して取り組むこと_」が必要です。
なんの目的意識も持たずにアプリ開発に取り組むことは、最大限の成長を期待できません。
このアプリ開発は上記に挙げた目的意識を達成し、自走力を身につけるために開発したものです。
